  <action>
    1. Create `src/lib/ai/generate-exercise.ts`:
       - `generateExercisePrompt(figure, difficulty, locale)`: build a prompt asking the AI to produce a short original text (3-5 sentences) that demonstrates the given figure at the given difficulty level
       - Call the AI via `generateText` with `Output.object` returning `{ text: string, figureSlug: string, answer: string }`
       - For identification exercises: the answer is the figure slug
       - For completion exercises: the answer is the missing word/phrase; mark the blank with `___`
       - Keep the same provider pattern (Groq primary, Gemini fallback)
    
    2. Create `src/app/api/ai/generate-exercise/route.ts`:
       - POST handler accepting `{ figureSlug, type: "identification" | "completion", difficulty: number, locale }`
       - Validates input, calls the generator, returns structured result
       - Returns 400 on invalid input, 500 on AI failure with safe fallback
    
    3. Update exercises page to surface generation:
       - Add a "Generar nuevo ejercicio" / "Generate new exercise" button to the exercise card
       - When clicked, calls the generate-exercise API and replaces the current exercise
       - Show loading state during generation
       - Keep it simple: one generate button per figure, replaces existing pre-seeded exercises
    
    4. Add locale strings for generation UI and feedback
  </action>
