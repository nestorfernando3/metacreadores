## Architecture

- **`generateExercisePrompt(figure, difficulty, type, locale)`** — builds system + user prompt. System prompt sets tutor role; user prompt specifies figure, difficulty vocabulary guidance, and output format. Identification prompts ask for a text + figure slug; completion prompts ask for a text with `___` blank + missing word.
- **`generateExercise(...)`** — calls `generateText` with `Output.object({ schema: GeneratedExerciseSchema })`. Uses `getDetectionConfig` (Groq) as primary, falls back to `getFallbackConfig` (Gemini). Returns `null` if both fail.
- **API route** — validates with Zod, looks up figure from DB via `getFigureBySlug`, calls generator, returns `{ text, answer, figureSlug, figureName, type }`. Returns 400/404/500 on errors.
- **UI** — each exercise card in the list view shows a "Generar nuevo" button. On click, calls the API, replaces the card's prompt/answer with the AI-generated version. Shows inline spinner during generation and a "Nuevo" badge on generated exercises. Identification exercises generated by AI (no multiple-choice options) render a free-text input.
