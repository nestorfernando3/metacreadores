**Key decisions from CONTEXT.md:**
- AI stack: Vercel AI SDK 5.x with Groq primary and Gemini fallback
- Behavior: tutor, never judge; no grades or scores
- Pattern: stateless multi-specialist passes (detection then feedback)
- Accuracy risk: hallucination + offset fidelity + regional Spanish variation

**AI contract:**
- Structured outputs must use Zod schemas
- Detection should return figure slug, offsets, confidence, explanation, and ambiguity notes
- Feedback should be separately generated from the structured detection result
- The model should only see the current text and a small set of figure candidates, not the full catalog
</context>
