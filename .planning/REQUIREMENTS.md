# Requirements: Metacreadores

**Defined:** 2025-05-04
**Core Value:** La IA como lector experto que analiza el texto del estudiante y le explica por qué una figura funciona o se siente forzada — retroalimentación tipo tutor, no calificación.

## v1 Requirements

### Catálogo de Figuras

- [ ] **CATALOG-01**: Estudiante puede ver definición clara y ejemplos de cada figura retórica
- [ ] **CATALOG-02**: Catálogo inicial incluye 15-20 figuras retóricas esenciales (metáfora, símil, hipérbole, personificación, aliteración, etc.)
- [ ] **CATALOG-03**: Ejemplos incluyen textos de literatura real, no solo ejemplos inventados
- [ ] **CATALOG-04**: Estudiante puede navegar y buscar figuras por nombre o tipo
- [ ] **CATALOG-05**: Cada figura incluye contexto histórico o literario de su uso

### Ejercicios

- [ ] **EXER-01**: Estudiante puede practicar identificación de figuras retóricas en textos
- [ ] **EXER-02**: Estudiante puede practicar completar figuras retóricas incompletas
- [ ] **EXER-03**: Estudiante recibe retroalimentación inmediata en ejercicios
- [ ] **EXER-04**: Dificultad de ejercicios se adapta al nivel del estudiante
- [ ] **EXER-05**: Ejercicios incluyen textos generados por IA adaptados al nivel del estudiante

### Escritura con IA

- [ ] **WRITE-01**: Estudiante puede escribir texto en un editor con formato básico
- [ ] **WRITE-02**: IA resalta figuras retóricas encontradas en el texto del estudiante
- [ ] **WRITE-03**: IA explica por qué una figura funciona o se siente forzada, en tono de tutor (no calificador)
- [ ] **WRITE-04**: IA sugiere figuras específicas que podrían mejorar el texto
- [ ] **WRITE-05**: IA ofrece análisis comparativo de alternativas ("qué pasaría si usaras X en vez de Y")

### Autenticación

- [ ] **AUTH-01**: Usuario puede registrarse con email y contraseña
- [ ] **AUTH-02**: Usuario puede iniciar sesión con email y contraseña
- [ ] **AUTH-03**: Sesión de usuario persiste entre navegaciones del navegador

### Progreso

- [ ] **PROG-01**: Estudiante puede guardar textos escritos en su cuenta
- [ ] **PROG-02**: Estudiante puede ver progreso de figuras que ha practicado
- [ ] **PROG-03**: Estudiante puede ver ejercicios completados

### Modo Docente

- [ ] **TEACH-01**: Docente puede ver progreso de estudiantes en su clase
- [ ] **TEACH-02**: Docente puede asignar ejercicios o figuras específicas a estudiantes

## v2 Requirements

### Progreso Avanzado

- **PROG-04**: Estudiante tiene niveles de maestría por figura retórica
- **PROG-05**: Sistema recomienda qué figura o ejercicio practicar siguiente

### Modo Docente Avanzado

- **TEACH-03**: Docente puede ver textos escritos por estudiantes
- **TEACH-04**: Docente puede configurar nivel de dificultad por clase

### Análisis de Textos Reales

- **REAL-01**: Estudiante puede pegar cualquier texto (canción, noticia, tweet) para analizar figuras retóricas
- **REAL-02**: Sistema ofrece biblioteca de textos literarios pre-cargados para análisis

## Out of Scope

| Feature | Reason |
|---------|--------|
| Calificación automática numérica | La IA es tutor, no juez. Calificación contradice el valor central del producto. |
| App móvil nativa | Web responsive cubre el 90% de casos de uso con menor inversión. |
| Catálogo exhaustivo (80+ figuras) | Empieza con 15-20 esenciales; expandir después de validar contenido. |
| Gamificación con puntos/badges | Riesgo de sobrejustificación — recompensas externas matan motivación intrínseca para escritura creativa. |
| Integración con LMS (Google Classroom, etc.) | Demasiado complejo para v1; los docentes pueden compartir enlaces directamente. |
